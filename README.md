# Network Scanner

> Scanner reseau multi-fonctions en Python avec Scapy.

## Fonctionnalites
- ARP Scan (decouverte d'hotes)
- Port Scan (TCP SYN, TCP Connect)
- Detection OS (fingerprinting)
- Export des resultats (JSON, CSV)

## Prerequis
- Python 3.8+
- Scapy
- Droits root/admin

## Disclaimer
**Usage educatif uniquement.** Ne scannez que les reseaux dont vous avez l'autorisation.
